Pour la simulation en C:
    Installer R (package 'r-base')
    Installer le package ggplot2 sur R ( "install.packages('ggplot2')" )
    Installer le lecteur PDF evince (Pour afficher les graphiques)

Pour la simulation en Python:
    Installer le module Tkinter (package 'python-tk')
    installer La bibliotheque de traitement d'image de Python (Python Imaging Library)
        "sudo pip install Pillow"
